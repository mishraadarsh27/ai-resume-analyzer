"# ai-resume-analyzer" 

## Database
Ensure TiDB connection string is set in DATABASE_URL.

## AI Model
The app uses Groq API. Set GROQ_API_KEY in .env.

## Running Locally
Run `run.bat` to launch the server.

## Key Features
- AI Resume Parsing
- Gap Analysis
- Timeline Roadmap
- Interview Prep

## Premium UI/UX Design
- Sleek dark mode
- Glassmorphism panels
- Keyframe animations

## Automated Testing
- Run test_ui.py to verify flows
- Run test_upload.py to verify uploads
